# \# BI Developer Assessment

# 

# \## Scenario

# 

# Atlas Fuel Services has asked you to perform an initial commercial review of its business.

# 

# You have access to a SQL Server database containing customer, transaction, product, network, account manager, target and customer service case data.

# 

# Your task is to explore the data using SQL, identify useful commercial insights, export the results of your analysis, and produce a short Power BI report for senior management.

# 

# There is no single correct answer. We are interested in how you approach the data, what questions you choose to ask, and how clearly you communicate your findings.

# 

# \---

# 

# \## Assessment Task

# 

# Using SQL, investigate the data and identify the insights that you believe would be most valuable to the Commercial Director.

# 

# Once you have completed your SQL analysis:

# 

# 1\. Export the outputs of the SQL queries you wish to use.

# 2\. Build a concise Power BI report using those exported outputs.

# 3\. Prepare to present your findings back to the interview panel.

# 

# \---

# 

# \## Suggested Areas to Explore

# 

# These are suggestions only. You do not need to cover all of them.

# 

# \- Customer performance

# \- Product performance

# \- Network performance

# \- Regional performance

# \- Account Manager performance

# \- Customer service activity

# \- Revenue, volume and margin trends

# \- Data quality observations

# \- Any other insight you believe is commercially important

# 

# \---

# 

# \## Presentation

# 

# Please be prepared to talk through:

# 

# \- What you investigated

# \- Why you chose those areas

# \- What SQL you used

# \- What your Power BI visuals show

# \- What surprised you

# \- What you would recommend to senior management

# \- What additional data you would want if this were a real project

# 

# \---

# 

# \## What We Are Assessing

# 

# \- SQL capability

# \- Data exploration

# \- Commercial thinking

# \- Power BI visualisation

# \- Data quality awareness

# \- Problem solving

# \- Communication
