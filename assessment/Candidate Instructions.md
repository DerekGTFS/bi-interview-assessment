# Candidate Instructions

Welcome to the Atlas Fuel Services BI Developer Assessment.

This assessment is designed to simulate a typical Business Intelligence task where you are required to understand an unfamiliar dataset, explore it using SQL and communicate your findings through Power BI.

---

## Database Connection

Connect to the SQL Server using the following details:

| Setting | Value |
|---------|------|
| Server | localhost |
| Username | TFS001 |
| Password | TFS!Password1! |
| Database | InterviewDB |

Connection Settings:

- Authentication: SQL Login
- Encrypt: Optional
- Trust Server Certificate: True

---

## Database Setup

Once connected, execute the following scripts in order:

1. database/01_CreateTables.sql
2. database/02_LoadData.sql

This will create and populate the assessment database.

---

## Assessment

Once the database has been created, open:

assessment/Assessment Brief.md

This contains the assessment task.

Good luck.
